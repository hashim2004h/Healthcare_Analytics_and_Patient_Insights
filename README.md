# Healthcare Appointment Analysis Dashboard

An interactive data analytics project that analyzes healthcare appointment patterns to identify attendance behavior, no-show trends, and key influencing factors such as age, gender, and waiting time. The project combines data preprocessing, analysis, and visualization to deliver actionable insights using Power BI.

---

## Key Analysis Areas

- Attendance vs No-show behavior  
- Demographic analysis (Age & Gender)  
- Impact of waiting time on attendance  
- Overall appointment distribution  

---

## Dashboard Features

- KPI cards for total appointments, attendance rate, and no-show rate  
- Interactive visuals for demographic insights  
- Waiting time impact analysis  
- Clean and user-friendly dashboard design  

---

## Insights Derived

- Overall Attendance Rate: **71.48%**  
- No-show Rate: **28.52%**  
- Adults show the highest attendance levels  
- Longer waiting times tend to reduce attendance probability  
- Gender-based differences observed in attendance patterns
